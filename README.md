# Personal Portfolio Website - Dylan 

A personal portfolio website built to showcase my skills.

**https://6a61e8911e9fc5a74c95f13c--dylan-louw.netlify.app**

## Features
- **Responsive Design:**- adapts to different screen sizes.
- **Dark theme:**- the burnt orange with the dark background is more easy on the users eyes.
- **Scroll animations:**- animations make the scrolling experimce smoother for the user by sliding sections in as the user scrolls.
- **Typing effect:**- on the profile tagline the text is animated to appear like it being typed in live.
- **Interactive hover:**- components like nav, project cards and buttons have hover animations.
- **Sections:**
  -Profile
  -About me
  -Experience
  -Projects
  -Contact me
